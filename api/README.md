# api

## Setup

1. Install dependencies
2. Build server
3. Set FIRESTORE_EMULATOR_HOST in package.json
4. Run server

## Access

The server will run on localhost:8080

## Usage

### Install dependencies

```
yarn install
```

### Build server

```
yarn build
```

### Run server

```
yarn start
```

### Run tests

```
yarn test
```

### Run tests with coverage

```
yarn test:coverage
```

### Run linters

```
yarn lint
```

### Run linters with autofix enabled

```
yarn lint:fix
```
